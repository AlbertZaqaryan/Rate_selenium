# Rate_selenium

## Overview  
`Rate_selenium` is a lightweight Python project that demonstrates how to use Selenium in an object-oriented structure to parse data from websites. The project includes examples of browser automation, navigation, and extraction of web elements.

## Features  
- Launch and control a browser using Selenium  
- Navigate through web pages  
- Extract targeted elements and information  
- Clean, object-oriented project structure (`parser.py`, `change.py`)  
- Easily extendable for more complex parsing tasks  

## Getting Started  

### 1. Clone the repository  
```bash
git clone https://github.com/AlbertZaqaryan/Rate_selenium.git
cd Rate_selenium
````

### 2. Create a virtual environment and install dependencies

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install selenium
```

### 3. Install WebDriver

Make sure you have the correct WebDriver (ChromeDriver, GeckoDriver, etc.) installed and added to your system PATH.

### 4. Run the scripts

```bash
python parser.py
```

or

```bash
python change.py
```

## Project Structure

```
Rate_selenium/
├── parser.py     — main parsing script  
├── change.py     — additional/alternative example  
└── README.md     — project documentation  
```

## Requirements

* Python 3.x
* Selenium
* ChromeDriver / GeckoDriver or another WebDriver
